## Introduction

The <AppName /> backend is a [Modularized Maven Java project](https://maven.apache.org/guides/mini/guide-multiple-modules.html) composed by the following modules:

### Main module
- [BPM Interface](http://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-interface)

### Backend points of access
- [BPM Interface Rest API](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-interface-rest-api)
- [BPM Interface Shell](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-interface-shell)

### BPM engine integration modules
- [BPM Engine Interface](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-engine-interface)
- [BPM Engine Camunda Client](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-engine-camunda-client)
- [BPM Engine Camunda External Service Client](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-engine-camunda-external-service-client)

### Utility modules
- [BPM Rest Client](https://github.com/mmctech/cis-innovation-bpm-platform/tree/main/mmc-bpm-rest-client)
