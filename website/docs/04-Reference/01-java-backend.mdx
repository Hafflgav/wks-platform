## Introduction

The <AppName /> backend is a [Modularized Maven Java project](https://maven.apache.org/guides/mini/guide-multiple-modules.html) composed by the following modules:

### Main module
- [Case Engine](https://github.com/wkspower/wks-platform/tree/main/case-engine)

### Backend points of access
- [Case Engine Rest API](https://github.com/wkspower/wks-platform/tree/main/case-engine-rest-api)

### BPM engine integration modules
- [BPM Engine Interface](https://github.com/wkspower/wks-platform/tree/main/bpm-engine-interface)
- [BPM Engine C7 Client](https://github.com/wkspower/wks-platform/tree/main/bpm-engine-c7-client)
- [BPM Engine C7 External Service Client](https://github.com/wkspower/wks-platform/tree/main/bpm-engine-c7-external-service)

### Utility modules
- [Rest Client](https://github.com/wkspower/wks-platform/tree/main/rest-client)
